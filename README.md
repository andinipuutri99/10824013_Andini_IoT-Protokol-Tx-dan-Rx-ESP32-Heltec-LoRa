LoRa Communication (Tx–Rx) with ESP Heltec + IoT Integration

Deskripsi:
Proyek ini merupakan implementasi komunikasi data berbasis LoRa (Long Range) 
menggunakan dua perangkat ESP Heltec LoRa WiFi V3 sebagai Transmitter (Tx) dan Receiver (Rx).

Sistem dirancang untuk:
- Mengirim data secara nirkabel menggunakan LoRa
- Menerima data pada perangkat Rx
- Menampilkan data pada Serial Monitor dan OLED

Tools:
- ESP Heltec LoRa WiFi V3 (2 unit)
- Kabel USB
- Laptop/PC
- Arduino IDE
- Library: RadioLib, U8g2lib, SPI, WiFi
